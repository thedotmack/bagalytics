## Architecture

Bagalytics is a Next.js 16 app that tracks creator fees for Bags.fm tokens on Solana. It displays real-time analytics for the 1% creator fee on token trading volume.

### Data Flow

1. **Frontend** (`src/app/page.tsx`) - Single-page client component that fetches token data and displays metrics/charts
2. **Token API** (`src/app/api/token/[address]/route.ts`) - Aggregates data from:
   - DexScreener API for price, volume, liquidity, and transaction data
   - Bags SDK for lifetime fees and token creator information
   - SOL price conversion for USD calculations

### External Dependencies

- **Bags SDK** (`@bagsfm/bags-sdk`) - Fetches lifetime creator fees and token creator data from Bags.fm on-chain state
- **DexScreener API** - Primary source for real-time token metrics (no API key required)

### Environment Variables

Required in `.env.local`:
- `SOLANA_RPC_URL` - Solana RPC endpoint (Chainstack, Helius, QuickNode, etc.)
- `BAGS_API_KEY` - Bags.fm API key for SDK access
- `REDIS_URL` - Redis connection string for caching

### UI Components

Custom lightweight UI components in `src/components/ui/` (Card, Button, Input, Alert). These are simplified versions without external dependencies like shadcn.
